

# 💰 Expense Tracker

A simple **Spring Boot web application** to manage daily expenses. Users can register, log in, add expenses, update them, and view a list of expenses.

---

## 🚀 Features
- 👤 User Registration & Login
- ➕ Add New Expense
- ✏️ Update Existing Expense
- 📋 View All Expenses
- 📂 JSP-based User Interface

---

## 🛠️ Tech Stack
- **Backend:** Java, Spring Boot (MVC + Services + Repositories)
- **Frontend:** JSP, HTML, CSS
- **Database:** (H2/MySQL – configure in `application.properties`)
- **Build Tool:** Maven
- **Version Control:** Git + GitHub
